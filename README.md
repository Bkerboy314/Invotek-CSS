# Invotek CSS Sheet
The stylesheet used for Invotek. The website is down so I had to use the version from the EternityDev community archive. (https://satellite9.com)

To install it, simply download this repo, then move styles.css and fonts to your website files. The readmy does not need to be copied.

Crediting me is optional.
